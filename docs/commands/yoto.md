## yoto

A CLI tool for managing Yoto cards and players

### Synopsis

YotoCLI is a tool for advanced users to manage their Yoto library.
It allows for uploading files, creating playlists, and managing device state directly from the terminal.

### Options

```
      --config string   config file (default is $HOME/.config/yotocli/config.yaml)
  -h, --help            help for yoto
```
