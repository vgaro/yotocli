## yoto edit

Edit properties of a playlist or track

### Synopsis

Modify the metadata of a playlist or track, such as the title, author, or description.

```
yoto edit <playlist[/track]> [flags]
```

### Examples

```
  # Rename a playlist
  yoto edit "Bedtime Stories" --name "Sleepy Time"

  # Update playlist metadata
  yoto edit "Sleepy Time" --author "Dad" --description "Read by Dad"

  # Rename a specific track
  yoto edit "Sleepy Time/1" --name "Chapter 1"
```
